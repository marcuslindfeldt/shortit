# ShortIt!

## Installation

```sh
npm install
```

## Client

```sh
npm run start
```

Production build of client:

```sh
npm run build
```

## Server

```sh
npm run serve
```

## Docker

```sh
docker build -t shortit
docker run -it -p 4000:4000 shortit
```

## Lint

```
npm run lint
```

## Tests

```
npm test
```
