# ShortIt!

## Installation

```sh
npm install
```

## Client

```sh
npm run start:client
```

Production build of client:

```sh
npm run build
```

## Server

```sh
npm start
```

## Docker

```sh
docker build -t shortit
docker run -it -p 4000:4000 shortit
```

## Lint

```
npm run lint
```

## Tests

```
npm test
```

## Deployment

Deploy to now.sh by downloading the now CLI and running `now` from the app root.

```
npm install -g now
now
```
